# ansible-zabbix-repo

Add zabbix repository

Works on:
* CentOS 6
* CentOS 7
* Ubuntu trusty
* Ubuntu xenial


## Variables
`zabbix_version: 3.0`


## Dependencies

## Example Playbook
    - hosts: all
      roles:
        - role: ansible-zabbix-repo
          zabbix_version: 3.2
